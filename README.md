# 数理金融期末习题和计算公式

习题和公式在main.ipynb

计算公式所需代码在./eqs文件夹下，每个文件是对应章节的代码

每个习题是文件中的一个markdown块，课本中对应公式也在此markdown块中，每个习题下面是调用对应代码的Python代码块

ipynb文件无法自动更新./eqs文件夹下的代码更改，首次打开需要执行文件的第一个Python代码块：

```
%load_ext autoreload
%autoreload 2
```
之后ipynb文件在每次执行引用代码时能更新./eqs文件夹下的代码

注意：本项目所有利率等类似变量，输入时不需要输入百分号

例如题目给定利率$r=1\%$，只需要在对应位置输入：

```
r=1
```

根据公式写计算代码时也注意这点
